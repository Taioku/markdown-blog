## Markdown Support

### Text Formatting
- **Bold text**
- *Italic text*
- ~~Strikethrough~~
- `Inline code`

### Headers
The application supports all header levels:

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

### Lists

**Unordered Lists:**
- Item 1
- Item 2
  - Nested item
  - Another nested item

**Ordered Lists:**
1. First item
2. Second item
3. Third item

### Code Blocks

```javascript
function hello() {
    console.log("Hello, World!");
}
```

```python
def greet(name):
    return f"Hello, {name}!"
```
